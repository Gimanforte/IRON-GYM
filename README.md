
# IRON GYM

Este projeto foi desenvolvido com o objetivo de criar a identidade visual de uma academia fictícia chamada Iron Gym, utilizando recursos de design moderno e impactante no Canva.


## Sobre o projeto
A proposta deste trabalho foi construir uma apresentação visual forte, voltada para o público que busca motivação, disciplina e evolução física.

O design foi pensado para transmitir:

Força

Determinação

Energia

Estilo moderno
## Objetivo
Desenvolver um material visual atrativo que represente uma academia profissional, podendo ser utilizado como base para divulgação, redes sociais ou apresentação institucional.
